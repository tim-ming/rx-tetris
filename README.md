## Controls

| Key             | Action                    |
| --------------- | ------------------------- |
| A \| ArrowLeft  | Move Left                 |
| D \| ArrowRight | Move Right                |
| ArrowUp         | Spin Right                |
| Z               | Spin Left                 |
| C               | Hold                      |
| S \| ArrowDown  | Soft Drop                 |
| Space           | Hard Drop                 |
| P               | Pause/Resume              |
| R               | Restart(only on game end) |
